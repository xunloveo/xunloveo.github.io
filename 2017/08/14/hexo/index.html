Use hexo ,next to build yourself blog

发表于 2017-08-14 | 分类于 hexo
字数统计 313 | 阅读时长 1

Welcome to Hexo! This is your very first post. Check documentation for more info. If you get any problems when using Hexo, you can find the answer in troubleshooting or you can ask me on GitHub.

安装next

你可以直接下载将文件拷贝到hexo下themes目录里，也可以在hexo目录下右键git bash通过命令：    
git clone https://github.com/iissnan/hexo-theme-next themes/next

启用主题

与所有 Hexo 主题启用的模式一样。 当 克隆/下载 完成后，打开 站点配置文件， 找到 theme 字段，并将其值更改为 next 。

验证

hexo generate 生成静态网页
hexo s 然后输入http://localhost:4000 就可以看到

配置

## 主题配置
    通过修改next主题下的_config.yml的scheme字段，配置不同的风格。
    # Schemes
    scheme: Muse
    #scheme: Mist
    scheme: Pisces 
    默认是Muse

## 菜单设置
    通过修改next主题下的_config.yml的menu字段，选定显示的菜单项。

    menu:
        home: /
        categories: /categories
        about: /about
        archives: /archives
        tags: /tags
        #commonweal: /404.html

    其中，home代表主页，categories代表分类页，about代表关于页面，archives代表归档页，commonweal代表404页面（page not found时候显示的页面）。

## 新建标签页面
    hexo new pages tags    
    输入命令后，在myBlog/source下会新生成一个新的文件夹tags，在该文件夹下会有一个index.md文件
